# üìö eBook System - Online Book Management

## üìù Description

This project is an **online book management system** (eBook system) that allows users to **browse**, **search**, **read**, **review**, and **purchase books** online. It provides functionalities for managing books, viewing book details, handling user interactions, and integrating payment options. The system is built using:

- **HTML** for structuring the web pages.
- **CSS** for styling and making the website visually appealing.
- **JavaScript** for adding interactivity.
- **PHP** for server-side functionality.
- **MySQL** for database management.
- **Bootstrap** for responsive design.
- **jQuery** for simplifying DOM manipulation.
- **WOW.js** and **GSAP** for animations.

---

## üöÄ Features

‚ú® **Browse Books**: Users can view a catalog of available books in the system.  
üîç **Search Books**: Quickly find books based on title, author, or genre.  
üìñ **Book Details**: View detailed information about each book, including the author, description, and availability.  
üí≥ **Purchase Books**: Users can pay using COD, MasterCard, and other supported payment methods.  
‚≠ê **Review Books**: Submit and read reviews for books.  
üë§ **User Management**: Users can register, log in, and manage their profiles.
